## Sprint 1: 基本診断機能とリード獲得フロー

### 実装する機能
- ドメイン入力フォーム: 企業ドメインを入力する画面
- 診断実行機能: SPF/DKIM/DMARCの設定をチェック
- 結果表示: 3つのカード形式で診断結果を表示
- CTA配置: 問題発見時に専門家相談へ誘導

### スプリント契約（完了条件）
以下の全条件を満たした場合のみ、このスプリントは完了とする。

#### 画面表示
- [ ] ページを開くと「メール配信診断ツール」のタイトルが表示される
- [ ] 「あなたの会社のメールは正しく届いていますか？」というキャッチコピーが表示される
- [ ] ドメイン入力欄に「例: example.com」のプレースホルダーが表示される
- [ ] 「診断する」ボタンが青色で目立つように配置されている
- [ ] スマートフォン（375px）、タブレット（768px）、デスクトップ（1440px）で正しくレイアウトされる

#### ドメイン入力と診断開始
- [ ] ドメイン入力欄に「test.com」と入力して診断ボタンをクリックできる
- [ ] 診断中は「診断中...」のローディング表示が3秒以内に開始される
- [ ] ローディング中は診断ボタンが無効化される
- [ ] 診断完了後に診断ボタンが再度有効化される
- [ ] 空欄で診断ボタンをクリックすると「ドメインを入力してください」のエラーが表示される
- [ ] 不正な形式（スペースのみ、日本語ドメイン以外の全角文字）でエラーメッセージが表示される

#### SPF診断結果
- [ ] SPFレコードが設定されている場合、緑色のチェックマークと「設定済み」が表示される
- [ ] SPFレコードが未設定の場合、赤色の警告アイコンと「未設定」が表示される
- [ ] SPF未設定時に「会社名で偽の請求書が送られ、Gmailに届かなくなり、商談を逃す可能性があります」の説明文が表示される
- [ ] 「SPFとは何ですか？」をクリックすると「会社の看板を勝手に使った偽メールをブロックする機能。郵便局で差出人をチェックするイメージ」という説明が展開される

#### DKIM診断結果
- [ ] DKIMレコードが設定されている場合、緑色のチェックマークと「設定済み」が表示される
- [ ] DKIMレコードが未設定の場合、オレンジ色の注意アイコンと「未設定」が表示される
- [ ] DKIM未設定時に「メールが途中で書き換えられ、振込先変更詐欺の危険があります」の説明文が表示される
- [ ] 「DKIMとは何ですか？」をクリックすると「メールに会社の実印を押す機能。割印のように改ざん防止する」という説明が展開される

#### DMARC診断結果
- [ ] DMARCレコードが設定されている場合、緑色のチェックマークと「設定済み」が表示される
- [ ] DMARCレコードが未設定の場合、黄色の情報アイコンと「未設定」が表示される
- [ ] DMARC未設定時に「CEO詐欺で中小企業でも数百万円〜数千万円の被害に遭うケースが急増しています」の説明文が表示される
- [ ] 「DMARCとは何ですか？」をクリックすると「SPFとDKIMを組み合わせた最終防衛ライン。社員証と入館証の二重チェック」という説明が展開される

#### CTAと誘導
- [ ] 1つ以上の項目が未設定の場合、画面下部に大きな「専門家に相談する」ボタンが表示される
- [ ] 相談ボタンの上に「設定には専門的な知識が必要です」のメッセージが表示される
- [ ] 相談ボタンをクリックすると外部の問い合わせフォーム（仮URL: https://forms.example.com/email-support）に遷移する
- [ ] 全て設定済みの場合は「問題ありません」と緑色のメッセージが表示される
- [ ] 全て設定済みでも「さらに詳しく診断したい方はこちら」のリンクが表示される

#### エラーハンドリング
- [ ] 存在しないドメイン「thisisnotarealdomainxyz123.com」で「ドメインが見つかりません」エラーが表示される
- [ ] ネットワークエラー時に「接続エラーが発生しました。しばらく待ってから再試行してください」が表示される
- [ ] DNS APIのレート制限時に「診断が混み合っています。少し待ってから再試行してください」が表示される

#### パフォーマンス要件
- [ ] 診断処理が3秒以内に完了する（正常なドメインの場合）
- [ ] Lighthouse Performance スコアが90以上
- [ ] Lighthouse Accessibility スコアが90以上
- [ ] Lighthouse Best Practices スコアが90以上
- [ ] First Contentful Paint (FCP) が1.8秒以下
