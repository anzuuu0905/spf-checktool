## Sprint 3: 複数ドメイン診断とFAQ

### 実装する機能
- 複数ドメイン一括診断: CSVアップロードまたはテキストエリアでの複数入力
- よくある質問（FAQ）: 初心者向けの丁寧な説明ページ
- 診断スコア表示: 総合的なメール配信健全性スコアの算出

### スプリント契約（完了条件）
以下の全条件を満たした場合のみ、このスプリントは完了とする。

#### 複数ドメイン診断
- [ ] 「複数ドメインを診断」タブが追加されている
- [ ] テキストエリアに複数ドメイン（改行区切り）を入力できる
- [ ] 最大10ドメインまで同時診断できる
- [ ] 11個以上入力すると「最大10個まで」のエラーが表示される
- [ ] CSVファイルをドラッグ&ドロップでアップロードできる
- [ ] 診断結果が表形式で一覧表示される
- [ ] 各行にドメイン名、SPF、DKIM、DMARC、総合評価が表示される
- [ ] 「CSVでダウンロード」ボタンで結果をエクスポートできる

#### FAQ セクション
- [ ] ヘッダーに「よくある質問」リンクが追加されている
- [ ] 「メールが届かない原因は？」の質問と回答が表示される
- [ ] 「SPF/DKIM/DMARCを設定するメリットは？」の質問と回答が表示される
- [ ] 「設定にはどれくらい時間がかかる？」の質問と回答が表示される
- [ ] 「自分で設定できますか？」の質問と回答が表示される
- [ ] 各回答の最後に「詳しくは専門家にご相談ください」のCTAが配置される
- [ ] アコーディオン形式で質問をクリックすると回答が展開される

#### 診断スコア機能
- [ ] 診断結果に0-100点の「メール配信健全性スコア」が表示される
- [ ] SPF設定済み: +40点、DKIM設定済み: +30点、DMARC設定済み: +30点で計算される
- [ ] 80点以上: 緑色で「優秀」バッジが表示される
- [ ] 50-79点: 黄色で「改善の余地あり」バッジが表示される
- [ ] 49点以下: 赤色で「要対策」バッジが表示される
- [ ] スコアの横に「業界平均: 65点」の比較情報が表示される
- [ ] 「スコアを上げる方法」をクリックすると改善提案が表示される
