## Sprint 2: 診断精度向上と詳細表示

### 実装する機能
- 診断履歴機能: 最近診断したドメインの一時保存と再表示
- 詳細診断モード: SPFレコードの内容解析と問題点の具体的指摘
- 診断結果のシェア: PDFダウンロードまたはリンク共有機能

### スプリント契約（完了条件）
以下の全条件を満たした場合のみ、このスプリントは完了とする。

#### 診断履歴
- [ ] 診断したドメインがブラウザのローカルストレージに保存される
- [ ] トップページに「最近の診断履歴」セクションが表示される（最大5件）
- [ ] 履歴から1クリックで再診断できる
- [ ] 履歴の各項目に診断日時が「2024年1月1日 12:00」形式で表示される
- [ ] 「履歴をクリア」ボタンで全履歴を削除できる

#### SPF詳細解析
- [ ] SPFレコードに「~all」が含まれる場合「ソフトフェイル設定」の注意が表示される
- [ ] SPFレコードに「-all」が含まれる場合「ハードフェイル設定」の情報が表示される
- [ ] includeが10個を超える場合「DNS参照制限に注意」の警告が表示される
- [ ] IPv4/IPv6アドレスが含まれる場合、その数が表示される
- [ ] 「もっと詳しく見る」をクリックすると生のSPFレコードが表示される

#### DKIM詳細解析
- [ ] 複数のDKIMセレクタを検査できる（default, google, amazonses）
- [ ] DKIMキーの長さが1024bit未満の場合「セキュリティ強度が低い」警告が表示される
- [ ] DKIMキーが2048bit以上の場合「推奨セキュリティレベル」が表示される
- [ ] セレクタが見つからない場合「一般的なセレクタで検出できませんでした」が表示される

#### DMARC詳細解析
- [ ] DMARCポリシーが「none」の場合「監視モードです」の情報が表示される
- [ ] DMARCポリシーが「quarantine」の場合「隔離設定です」の警告が表示される
- [ ] DMARCポリシーが「reject」の場合「拒否設定です」の情報が表示される
- [ ] レポート送信先（rua/ruf）が設定されているか表示される
- [ ] pctパラメータの値が100未満の場合「段階的適用中」が表示される

#### 診断結果の共有
- [ ] 「診断結果をダウンロード」ボタンでPDF形式の簡易レポートが生成される
- [ ] PDFには診断日時、ドメイン名、3項目の診断結果が含まれる
- [ ] PDFのファイル名が「メール診断_example.com_20240101.pdf」形式になる
- [ ] 「診断結果をシェア」ボタンでユニークなURLが生成される
- [ ] シェアURLを開くと同じ診断結果が表示される（7日間有効）
