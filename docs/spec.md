# メール配信診断ツール

## 概要
中小企業経営者向けのメール配信設定診断ツール。ドメイン名を入力するだけでSPF/DKIM/DMARCの設定状況を診断し、問題がある場合は専門家への相談を促すリードジェネレーションツール。

## ターゲットユーザー
- 税理士のクライアント企業（従業員50名以下の中小企業経営者）
- 「メールが届かない」「迷惑メールに入る」問題を抱えている
- IT専門知識がない、SPF/DKIM/DMARCという言葉を初めて聞くレベル

## 機能一覧
### 優先度: 高
- ドメイン入力による簡単診断
- SPF/DKIM/DMARC設定の自動チェック
- 初心者向けの分かりやすい診断結果表示
- 問題発見時の専門家相談への誘導（CTA）
- レスポンシブ対応（スマホ/タブレット/PC）

### 優先度: 中
- 診断履歴の一時保存（ブラウザストレージ）
- 診断結果のPDFダウンロード
- よくある質問（FAQ）セクション
- 複数ドメインの一括診断

### 優先度: 低
- メール通知機能（診断結果の送信）
- 定期診断スケジュール機能
- APIとしての外部提供
- 多言語対応（英語版）

## 初心者向け説明文（確定版）

本ツールで使用する説明文は、IT知識ゼロの中小企業経営者向けに最適化されています。
詳細は `/docs/beginner-explanations.md` を参照。

### SPF（差出人チェック機能）
- **一言説明**: 会社の看板を勝手に使った偽メールをブロックする「差出人チェック機能」
- **設定していないとどうなる？**: 会社名で偽の請求書が送られ、Gmailに届かなくなり、商談を逃す
- **設定するとどうなる？**: 「この郵便局から送られる」証明書で偽物をブロック
- **比喩**: 受付の社員リストで本物チェック

### DKIM（会社の実印機能）
- **一言説明**: メールに「会社の実印」を押して本物であることを証明
- **設定していないとどうなる？**: 振込先変更詐欺で改ざんされる危険
- **設定するとどうなる？**: デジタル会社印で改ざん防止
- **比喩**: 割印を押した改ざん防止シール

### DMARC（最終防衛ライン）
- **一言説明**: SPFとDKIMを組み合わせて偽メールを完全シャットアウト
- **設定していないとどうなる？**: CEO詐欺で数百万円〜数千万円の被害
- **設定するとどうなる？**: 偽物を徹底的にブロック
- **比喩**: 社員証と入館証の二重チェック

## 技術スタック推奨
- **フロントエンド**: 静的HTML + Vanilla JavaScript + CSS
- **デザインシステム**: Cursor風デザイン（`docs/design.md` を適用）
- **DNS検証**: Google Public DNS API（`dns.google.com`）を使用
- **テスト**: Vitest（単体テスト）+ Playwright（E2Eテスト）
- **ビルドツール**: Vite
- **デプロイ**: Netlify または Vercel（静的ホスティング）
